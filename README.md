# Alarm Clock
  ## บทคัดย่อ
        เป็นการทดลองทำนาฬิกาปลุก ซึ่งสามารถตั้งเวลาปัจจุบัน และเวลาปลุกได้ มีจอ LCD แสดงเวลาและ switch สำหรับตั้งเวลา ทำงานโดยจะมีการตั้งเวลาปัจจุบันก่อน และตั้งเวลาปลุก เมื่อถึงเวลาจะมีเสียงเตือนดังขึ้น 
        
  ## รายละเอียด
        ใช้โปรแกรม Arduino ไว้สำหรับการตั้งเวลาปลุกด้วย Microcontroller ที่ทำงานด้วย Arduino เชื่อมกับ Battery 9V , Buttons , Buzzer , DS3231 RTC และ LCD ที่มี Output เป็นการแสดงเวลาในจอ LCD โดยใช้ Arduino เชื่อมกับ Battery ขนาด 9 โวลต์ ที่ช่อง Vn ในส่วนของ Power และต่อปุ่มต่างๆ รวมไปถึง DS3231 RTC ในช่อง SCL SDA เข้ากับ Analog Input เพื่อที่จะใช้ปุ่มในการตั้งค่าต่างๆ และมี Buzzer เพื่อเป็น Alarm กับ LCD เพื่อแสดงผล Output
